About Me

I started my career over 10 years ago as a frontend web developer (html, css, js) and very quickly I learned the difference between web development vs software engineering. I had a desire to work with the backend application code and databases.

I took on side projects learning PHP and MySQL. I've gained extensive experience with open source CMS's such as WordPress, Drupal, Pimcore and ezPublish. I also spent a lot of time learning and developing web apps with the latest and greatest PHP frameworks such as ZendFramework, Symfony, Slim and Expressive.

Cloud computing became popular and it felt like I became a operations engineer over night. I learned a lot about system administration, networking and DNS. I learned how to build scalable, fault-tolerant, robust and highly-available applications on Amazon AWS. I eventually became an AWS Certified Solutions Architect.
